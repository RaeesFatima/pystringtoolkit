# PyStringUtils

Simple Python library with handy string utilities like:
- `to_snake_case()`
- `to_camel_case()`
- `to_upper_case()`
- `to_lower_case()`
- `to_pascal_case()`
- `to_kebab_case()`

## Installation
pip install pystringutils
## Example
```python
from pystringutils import to_snake_case

print(to_snake_case("Hello World!"))  # hello_world
